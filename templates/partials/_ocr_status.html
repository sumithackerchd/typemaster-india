Installation guide
Ubuntu / Debian 
sudo apt-get update
sudo apt-get install -y tesseract-ocr tesseract-ocr-hin poppler-utils
macOS (Homebrew) 
brew install tesseract tesseract-lang poppler
Windows 
# 1. Install Tesseract (UB Mannheim build):
#    https://github.com/UB-Mannheim/tesseract/wiki
# 2. Install Poppler and add its /bin to PATH:
#    https://github.com/oschwartz10612/poppler-windows/releases
# 3. Add Tesseract to PATH, or set in your environment:
#    TESSERACT_CMD=C:\Program Files\Tesseract-OCR\tesseract.exe

The Python packages (pytesseract, pdf2image, opencv-python-headless, Pillow) are already included in requirements.txt — only the system binaries above are needed. Restart the server after installing.
